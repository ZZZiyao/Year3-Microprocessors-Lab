# Microprocessors (施工中><)
Repository for Physics Year 3 microprocessors lab (collaborated with my lab partner [jy3122](https://github.com/jy3122))

A simple assembly program for PIC18 microprocessor, that counts to 100, putting the current count value out onto PORTB

Please refer to other branches for more sophisicated works.
